Mango Village


Example clip showcasing some game mechanics implemented.


Looks simple, but there's a lot going on in the background here!

  • Multiplayer with a client/server architecture, using durian as the networking library. All the movement and physics is being calculated on the server and sent to the client for rendering.
  • Physics system, powered by bevy_rapier, an impulse based physics simulator. The player is modeled as a character controller, with a custom collision system.
  • Pan-orbit camera system with 3d controls, and a locked follow camera mode on the player
  • Meshes are generated as .gltf/.glb files from Blender models
  • All built on top of the Bevy game engine under an Entity Component System (ECS) architecture.
